사진을 보면 ex03 디렉토리에 Exam0100.java를 컴파일 하고 실행을 할것이다

처음에

```java
javac -d bin/main src/main/java/com/eomcs/lang/ex03/Ex0100.java
```

로 입력하여 bin폴더의 main안에 src/main/java/com/eomcs/lang/ex03/Ex0100.java를 컴파일한 파일을 넣는다.

그리고 바이트코드로 생성된 파일을 실행하려면

```java
java -cp bin/main com.eomcs.lang.ex03.Exam0100
```

으로 실행하면 터미널에 출력을 한다.
